# JSON Formatter & Validator

A free online tool to validate, format, and minify JSON data instantly.

## Features

* JSON Validation
* JSON Formatting
* JSON Minification
* Copy Output
* Upload JSON File
* Download JSON File
* Dark / Light Mode
* Responsive Design

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* Vercel
